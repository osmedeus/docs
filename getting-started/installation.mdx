### Prerequisites

- Go **1.21** or later
- Git
- Make

### Build

```bash
# Clone the repository
git clone https://github.com/osmedeus/osmedeus.git
cd osmedeus

# Build the binary
make build

# Binary is created at bin/osmedeus
./bin/osmedeus --help

# Install to $GOBIN (or $GOPATH/bin)
make install
```

### Cross-Platform Builds

```bash
# Build for all platforms (linux, darwin, windows)
make build-all

# Output in bin/ directory:
# - osmedeus-linux-amd64
# - osmedeus-darwin-amd64
# - osmedeus-darwin-arm64
# - osmedeus-windows-amd64.exe
```

## Binary Tools Installation

Osmedeus can install security tools from its registry.

### Direct Fetch Mode

Downloads pre-built binaries:

```bash
# List available binaries
osmedeus install binary --list-registry-direct-fetch

# Install specific binaries
osmedeus install binary --name nuclei
osmedeus install binary --name nuclei --name httpx --name ffuf

# Install all required binaries
osmedeus install binary --all

# Include optional binaries
osmedeus install binary --all --install-optional

# Check installation status (without downloading)
osmedeus install binary --name nuclei --check
osmedeus install binary --all --check
```

### Nix Mode

Uses Nix package manager for reproducible builds:

```bash
# Install Nix package manager (if not present)
osmedeus install binary --nix-installation

# List Nix-compatible binaries
osmedeus install binary --list-registry-nix-build

# Install via Nix
osmedeus install binary --nix-build-install
osmedeus install binary --name nuclei --nix-build-install
```

### Add to PATH

```bash
# Add binaries folder to shell PATH
osmedeus install env

# Restart shell or source config
source ~/.bashrc  # or ~/.zshrc
```

## Validate Installation

```bash
# Check environment health
osmedeus install validate

# Show version
osmedeus version

# List workflows (confirms base folder setup)
osmedeus workflow list
```

## Directory Structure

After installation, the base folder (`~/osmedeus-base/`) contains:

```
~/osmedeus-base/
├── osm-settings.yaml      # Main configuration
├── binaries/              # Installed tools
├── data/                  # Wordlists, templates
├── workflows/             # Workflow YAML files
│   ├── modules/           # Module workflows
│   └── flows/             # Flow workflows
├── workspaces/            # Scan output
└── database-osm.sqlite    # SQLite database
```

## Install Workflows

Install workflows from external sources:

```bash
# From Git repository
osmedeus install workflow https://github.com/example/osmedeus-workflows.git

# From ZIP URL
osmedeus install workflow https://example.com/workflows.zip

# From local file
osmedeus install workflow ~/downloads/custom-workflows.zip
```

## Install Base Folder

Replace the entire base folder (backs up and restores database):

```bash
osmedeus install base https://example.com/osmedeus-base.zip
```

## Troubleshooting

### Binary not found

```bash
# Ensure binaries folder is in PATH
echo $PATH | grep osmedeus-base/binaries

# Add to PATH manually
export PATH="$HOME/osmedeus-base/binaries:$PATH"

# Or use osmedeus install env
osmedeus install env && source ~/.bashrc
```

### Permission denied

```bash
# Make binary executable
chmod +x ~/osmedeus-base/binaries/*
```

### Database issues

```bash
# Reset database
rm ~/osmedeus-base/database-osm.sqlite
osmedeus workflow list  # Recreates database
```
