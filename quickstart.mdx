<Check>Currently supports Linux and macOS natively. Windows users please use WSL.</Check>

<Steps>
  <Step title="Install the latest version of Osmedeus">

  ```bash Install the Osmedeus Binary
  curl -sSL https://docs.osmedeus.org/install.sh | bash
  ```

  </Step>
  <Step title="Fetch the latest version">
  ```bash
  osmedeus install
  ```
  </Step>
  <Step title="Run your first workflow">
  ```bash
  osmedeus run -m subdomain-enum -t example.com
  ```
  </Step>
</Steps>


See [CLI Reference](/cli) for more details.
