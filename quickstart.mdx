## Step 1: Install the latest version of Osmedeus

<Tabs>
  <Tab title="Navtive Installation (Recommended)">
  ```bash
  curl -sSL http://host.docker.internal:8888/install.sh | bash
  ```
  </Tab>
  <Tab title="Homebrew Installation" icon="leaf">
  ```bash
  curl -sSL http://host.docker.internal:8888/install.sh | bash
  ```
  </Tab>
  <Tab title="Windows">
    <Check>Currently supports Linux and macOS natively. Windows users please use WSL.</Check>
  </Tab>
</Tabs>

<Frame caption="Installation Osmedeus Core Engine">
  <img src="/images/installation/install-native.png" />
</Frame>

## Step 2: Install the default workflows

<Tabs>
  <Tab title="Use default preset workflows">
  ```bash
  osmedeus install base --preset
  ```
  </Tab>
  <Tab title="Install custom workflows from Git repository">
  ```bash
  osmedeus install base https://github.com/osmedeus/osmedeus-base.git
  ```
  </Tab>
</Tabs>


## Step 3: Validate the installation
```bash
osmedeus health
```

<Frame caption="Installation Validation">
  <img src="/images/installation/install-validation.png" />
</Frame>


## Step 4: Run your first workflow
```bash
osmedeus run -m subdomain-enum -t example.com
```
